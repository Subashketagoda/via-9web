# VIA9 Restaurant Website

This is a premium, responsive website for the **VIA9 Italian Restaurant**.

## How to View
Simply double-click `index.html` to open it in your default web browser.

## Features
- **Premium Design**: Dark theme with gold accents, perfectly suited for an elegant Italian dining experience.
- **Responsive**: Works beautifully on desktops, tablets, and mobile phones.
- **Interactive**: 
  - Smooth scrolling navigation.
  - Scroll-triggered animations for menu items and reviews.
  - Mobile-friendly hamburger menu.
  - Reservation form simulation (shows a success message).
- **Performance**: usage of CSS gradients and icons ensures fast loading times without large image files.

## Files
- `index.html`: Main website structure.
- `styles.css`: All visual styling and animations.
- `script.js`: Logic for navigation and interactivity.
- `favicon.svg`: Website icon.

Enjoy your new website!
